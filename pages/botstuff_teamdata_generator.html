🎮 SourceMod 队伍数据转换器

自动从现有代码提取队伍信息并生成优化的数据结构

使用说明：将你的现有SourceMod队伍代码粘贴到下方文本框中，工具会自动提取队伍信息并生成优化的代码。支持别名处理和数百个队伍的批量处理。

📥 输入现有代码

🎯 生成的 SourceMod 代码

📊 队伍数据文件

⚙️ 配置说明

文件结构：

addons/sourcemod/configs/teams.cfg

数据格式：

"Teams" { "Falcons" { "aliases" "Flcn" "players" "kyousuke,m0nesy,TeSeS,kyxsan,Niko" "logo" "fal" } }

使用方法：

  1. 将生成的SourceMod代码替换你的现有代码
  2. 将数据文件保存为 addons/sourcemod/configs/teams.cfg
  3. 重新编译并加载插件
